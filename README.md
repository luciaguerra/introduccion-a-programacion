## Trabajo práctico 2:
Realicé un juego con un personaje hecho solo con figuras geométricas, que cuando tratara de salir de sus límites perdiera una vida hasta tener que reiniciar el juego. Para este trabajo se utilizaron distintos eventos propios de Processing para permitir la interacción del usuario con el personaje y que este pudiera moverse a través de la pantalla, utilizando el teclado o el mouse pulsando los botones en pantalla.
